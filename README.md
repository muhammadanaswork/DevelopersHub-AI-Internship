# DevelopersHub AI/ML Internship Tasks
This repository contains my tasks for the AI/ML Engineering Internship at DevelopersHub Corporation.

**Phase 1**
* **Task 1: Exploring and Visualizing a Simple Dataset:** Conducted EDA on the Iris dataset using Pandas, Seaborn, and Matplotlib. 
* **Task 2: Predict Future Stock Prices:** Built a Linear Regression model using `yfinance` to predict Apple's short-term stock prices based on historical features.
* **Task 3: Heart Disease Prediction:** Built a Decision Tree classification model to predict cardiovascular risk, performed EDA, and evaluated the model using `ROC-AUC`, a confusion matrix, and feature importance.
* 
**Phase 2**
* **Task 2: End-to-End ML Pipeline:** Built a production-ready Scikit-Learn pipeline for predicting Telco customer churn, utilizing ColumnTransformer, GridSearchCV, and exported the model using `joblib`.
